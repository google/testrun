## Accessibility

Testrun is designed with accessibility at its core. We have ensured the application provides full support for the following:
 - Screen readers
 - Keyboard navigation
 - Responsize resizing & scaling
 - [Helperbird](https://www.helperbird.com/)

For a more thorough explanation of these features, download the [accessibility video here](accessibility.mp4).

If there are further accommodations you require when using Testrun, please raise an [issue here](https://github.com/google/testrun/issues/new/choose).
